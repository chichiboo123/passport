너는 뛰어난 UI/UX 감각을 지닌 시니어 프론트엔드 개발자야. 전 세계 어린이들이 자신의 '캐릭터(마스코트)'를 가상으로 여행 보낼 때 사용할 수 있는 '캐릭터 여행 여권(Character Travel Passport) 만들기' 웹앱을 개발해 줘. 

서버 배포 없이 열악한 네트워크의 교실 환경에서도 쉽게 실행할 수 있도록 반드시 **단일 HTML 파일(Single HTML: 내부에 CSS, JS 모두 포함)**로 작성해야 해. 

아래의 기획 및 디자인 가이드라인을 엄격하게 준수하여 코드를 작성해 줘.

### 1. 디자인 및 UI/UX 가이드라인
*   **전체적인 UI 콘셉트:** **CANVA(캔바) 앱과 같이 직관적인 에디터 인터페이스**로 구성해. 왼쪽에는 입력용 사이드바(컨트롤 패널)를 두고, 오른쪽에는 넓은 캔버스(미리보기)를 배치해.
*   **UI/UX 원칙:** 대한민국 정부 웹 UI/UX 가이드라인(KRDS)을 엄격히 준수해. 명확한 정보 계층, 접근성 높은 색상 대비, 깔끔한 폼 컨트롤(Input, Select), 명확한 포커스 및 호버 상태를 적용해.
*   **테마 색상:** 4가지 테마 컬러(블루, 레드, 그린, 브라운)를 제공하고, 사용자가 선택하면 사이드바 헤더와 여권 표지/속지 포인트 컬러가 즉각 변경되도록 해.
*   **폰트 설정:** 
    *   타이틀 및 여권 내 주요 강조: **'Black Han Sans'** (구글 웹폰트 CDN)
    *   본문 및 UI 텍스트: **'Pretendard GOV'** (웹폰트 CDN)
*   **아이콘:** 구글 머테리얼 심볼(Material Symbols Outlined) CDN을 사용해.

### 2. 화면 구성 요소 (에디터 & 캔버스)
*   **헤더(Header):** 앱 타이틀, 언어 선택 드롭다운(한국어, English, 日本語, Bahasa Indonesia).
*   **사이드바(Editor):**
    1.  여권 테마 선택 (4가지 색상 칩)
    2.  캐릭터 정보 입력: 사진 업로드, 이름, 탄생일(input date), 소속/국적, 좋아하는 것
    3.  스탬프 추가 폼: 이모지 입력창, 여행 장소, 방문 날짜, '스탬프 찍기' 버튼
    4.  데이터 관리: 'JSON 저장', 'JSON 불러오기' 버튼
*   **캔버스(Workspace):**
    *   상단에 '스탬프 페이지 보기/프로필 페이지 보기' 토글 버튼과 '이미지 저장' 버튼 배치.
    *   **프로필 모드:** 
        *   왼쪽 페이지: "PLEASE LOOK AFTER THIS TRAVELER"라는 텍스트가 적힌 여행 태그(Tag) 디자인 요소 배치.
        *   오른쪽 페이지: 캐릭터 사진, 입력된 정보, 그리고 가상의 MRZ 코드.
    *   **스탬프 모드:** 
        *   속지 배경(VISAS 워터마크) 위에 사용자가 추가한 이모지 스탬프들이 렌더링되는 영역.
*   **푸터:** 화면 우측 하단에 `<a href="https://litt.ly/chichiboo" target="_blank">Created by. 교육뮤지컬 꿈꾸는 치수쌤</a>` 텍스트 배치.

### 3. 핵심 기능 및 자바스크립트 로직
*   **상태 관리(State Management):** 모든 입력 데이터와 스탬프 배열, 선택된 언어 등을 중앙 상태 객체로 관리하고 실시간으로 캔버스에 렌더링해.
*   **다국어 지원(i18n):** 외부 라이브러리 없이 JS 내부에 한국어, 영어, 일본어, 인도네시아어 4개국어 번역 객체를 만들어 적용해. (기본어: 한국어)
*   **이모지 스탬프 로직:** 
    *   스탬프를 추가하면 속지 페이지의 랜덤한 위치와 각도로 도장(스탬프 테두리 + 이모지 + 장소 + 날짜)이 찍히도록 해. 색상은 5가지 색상 중 랜덤.
    *   **스탬프 삭제:** 캔버스에 찍힌 스탬프에 마우스를 올리면 삭제 아이콘('X' 또는 'DELETE')이 나타나고, 클릭 시 해당 스탬프가지워지도록 구현해.
*   **데이터 저장 및 복원:** 
    *   **JSON Export/Import:** 현재 작업 상태(사진 데이터 포함)를 JSON 파일로 다운로드하고, 다시 업로드하면 모든 상태가 복원되는 기능을 완벽히 구현해.
    *   **이미지 다운로드:** `html2canvas` 라이브러리를 사용해 캔버스의 여권 그래픽 영역을 PNG 이미지로 저장하는 기능.

### 4. 안전 및 법적 가이드라인
*   **로컬 처리:** 사진 업로드를 포함한 모든 데이터는 `FileReader`와 브라우저 메모리 내에서만 처리해 서버 전송을 원천 차단해.
*   **위조 방지:** 캔버스 배경에 사선으로 "NOT AN OFFICIAL DOCUMENT", "KIDS PLAY PASSPORT"라는 텍스트 워터마크를 투명하게 깔아둬.
*   실제 여권과 혼동되지 않도록 캐릭터의 신분증 느낌을 주는 레이아웃(Traveler 컨셉)을 유지해.

주석을 꼼꼼하게 작성하고, 디자인 퀄리티가 당장 서비스해도 될 만큼 훌륭하고 세련된 단일 HTML 코드를 코드 블록 하나로 출력해 줘.
